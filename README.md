# 💻 Loan Prediction App

A Machine Learning based web application that predicts whether a loan will be approved or not based on user inputs.

---

## 🚀 Features
- Predict loan approval instantly
- Simple and user-friendly interface
- Built using Machine Learning model
- Takes user inputs like income, credit history, etc.

---

## 🛠️ Tech Stack
- Python 🐍
- Pandas & NumPy
- Scikit-learn
- Streamlit

---

## 📊 How it Works
1. User enters details like:
   - Income
   - Loan Amount
   - Credit History
2. Model processes the data
3. Predicts:
   - ✅ Loan Approved  
   - ❌ Loan Not Approved
